## 👋 Welcome to homepage 🚀

Homepage - Application dashboard

## 📋 Description

A modern, fully static, fast, secure application dashboard with integrations for over 100 services.

## 🚀 Services

- **app**: Homepage (`ghcr.io/gethomepage/homepage:latest`)

## 📦 Installation

```shell
composemgr install homepage
```

## 🔧 Configuration

Edit files in `rootfs/config/homepage/`:
- `services.yaml` - Your services and bookmarks
- `widgets.yaml` - Dashboard widgets
- `settings.yaml` - General settings

## 🌐 Access

- **Homepage**: http://localhost:3000

## 📂 Volumes

- `./rootfs/config/homepage` - Configuration files
